Your Personal CryptoCoin Bank

Required setup

You should update pip and install project requirements.

easy_install --upgrade pip
easy_install base58
pip install -r requirements.txt

GeoIP

Install the GeoCity C library. You can get it from MaxMind.

On Mac OS X, you can install using homebrew:

brew install geoip

or MacPorts:

sudo port install libgeoip

Download GeoIP City datasets in binary format from MaxMind uncompress and put them in .geoip/ folder.

cd .geoip
wget http://geolite.maxmind.com/download/geoip/database/GeoLiteCity.dat.gz
gzip -d GeoLiteCity.dat.gz

Configuration

Copy template file mybitbank/settings-template.py into mybitbank/settings.py.

Copy teamplate file connections/config-template.py into connections/config.py
